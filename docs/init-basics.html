[SYSTEM://OPERADORES] ⚔️

🎯 Operadores Aritméticos

int a = 10, b = 5; int soma = a + b; // 15 int subtracao = a - b; // 5 int multiplicacao = a * b;// 50 int divisao = a / b; // 2 int resto = a % b; // 0

🔄 Operadores de Comparação

bool igual = (a == b); // false bool diferente = (a != b); // true bool maior = (a > b); // true bool menor = (a < b); // false bool maiorIgual = (a >= b); // true bool menorIgual = (a <= b); // false

⚡ Operadores Lógicos

bool x = true, y = false; bool e = x && y; // AND (false) bool ou = x || y; // OR (true) bool nao = !x; // NOT (false)
