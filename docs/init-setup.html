1. .NET SDK

🪟 Windows

# Verificar instalação atual dotnet --version # Opção 1: Usando winget winget install Microsoft.DotNet.SDK.7 # Opção 2: Download direto # Visite: https://dotnet.microsoft.com/download

🐧 Linux

# Ubuntu/Debian wget https://packages.microsoft.com/config/ubuntu/22.04/packages-microsoft-prod.deb -O packages-microsoft-prod.deb sudo dpkg -i packages-microsoft-prod.deb sudo apt update sudo apt install dotnet-sdk-7.0 # Fedora sudo dnf install dotnet-sdk-7.0 # Arch Linux sudo pacman -S dotnet-sdk # Verificar instalação dotnet --version
