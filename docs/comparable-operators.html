Operadores de Comparação

Servem para fazermos comparações e temos dois possiveis estados: verdadeiro ou falso.

Os operadores são:

  • == - Igual a;

  • != - Diferente de;

  • < - Menor que;

  • > - Maior que;

  • <= - Menor ou igual a;

  • >= - Maior ou igual a.
